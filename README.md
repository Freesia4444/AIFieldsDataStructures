# 🤖 AI Fields Explorer

C# ile yazılmış, yapay zeka alt alanları verisi üzerinde çeşitli veri yapıları ve algoritmaları uygulayan bir proje.

## 📌 Ne Yapıyor?

- Yapay zeka ana alanlarını (Machine Learning, NLP, Computer Vision vb.) ve alt alanlarını BST ile tutuyor
- En derin alt alan ağacını buluyor
- Tanım ve uygulama metinlerinden kelime frekans analizi yapıyor (Regex + BST)
- Ana alanları HashTable'a ekleyip güncelleme işlemi yapıyor
- MinHeap ile alanları alfabetik sıraya göre düzenliyor
- Bubble Sort ve Quick Sort'u Stopwatch ile karşılaştırıp hız farkını ölçüyor

## 🛠️ Kullandıklarım

- C# / .NET 8
- `BST<T>` — kendi yazdığım generic Binary Search Tree
- `MinHeap` — kendi yazdığım min yığını
- `Dictionary` — HashTable işlemleri için
- `Regex` — kelime ayrıştırma için
- `Stopwatch` — algoritma süresi ölçümü için
- Bubble Sort ve Quick Sort — kendi implementasyonum

## 🧠 Veri Yapıları

| Yapı | Kullanım Amacı |
|---|---|
| BST | Ana ve alt alan hiyerarşisi |
| MinHeap | Alfabetik sıralama |
| HashTable | Alan adı güncelleme |
| BST (kelime) | Frekans analizi |

## 🚀 Çalıştırmak İçin
```bash
dotnet run
```
